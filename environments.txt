- Python: >= 3.11
- Package manager: pip (latest recommended)
- Virtual environment: venv recommended

4. Local Setup (Recommended)
Step 1: Create and activate venv
- Windows PowerShell:
  python -m venv .venv
  .\.venv\Scripts\Activate.ps1

Step 2: Install project
- pip install -e .

Step 3: Prepare env file
- Copy .env.example to .env and adjust values.

Step 4: Start service
- uvicorn app.main:app --reload --host 0.0.0.0 --port 8010

5. Service Endpoints
- Home: http://127.0.0.1:8010/
- OpenAPI: http://127.0.0.1:8010/docs
- Health: http://127.0.0.1:8010/health

6. Data and Persistence Notes
- Persistent: SQLite data, ontology graph JSON, version snapshots, security audit log.
- In-memory (reset on restart): approval tickets runtime mapping and active branch runtime state.

7. Smoke Validation
- Python syntax check:
  python -m compileall -q app

- API quick check examples:
